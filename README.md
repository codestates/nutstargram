# nutstargram

App안의

- nav(데이터 없음)
  -data(나는 users,contents도 있다.)
  - logout
  - mypage
    - edit contents (contents 받아서 수정)
    - edit userinfo (users 받아서 편집)
      - private info (users.password 데이터 받아서 편집)
  - main (데이터 없다 props로 내려받아서 보여주는 역할)
  - write (데이터 쿼리날림 contents데이터 만드는 역할)
- login
  - sign up
